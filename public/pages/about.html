What is DevOps?

DevOps is the combination of development and operations — a set of practices that brings teams together to deliver software faster, safer, and more reliably through automation and collaboration.

About This Project

This DevOps Portal is a fully functional demonstration project built to showcase a complete end-to-end DevOps workflow. A static website (this one!) is containerized with Docker, pushed to AWS ECR, and deployed automatically on an EC2 instance via Jenkins CI/CD whenever code is pushed to GitHub.

The site itself is your interactive learning hub — with live pipeline visualization, real-time dashboards, and architecture diagrams. Perfect for portfolios and interviews.
